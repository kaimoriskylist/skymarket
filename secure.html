MFA Secure Guide

This guide explains how to fully secure your Microsoft / Minecraft account using MFA. Follow each step carefully.

Changing the Email

  1. Login to the account.
  2. Go to account.live.com/proofs/manage and enter the provided email.
  3. If prompted with "Enter the code we sent to se***@nethermfas.com", click Other ways to sign in or Use password instead.

Recover Your Password

  1. On the "Enter Password" screen, click Forgot password?
  2. Select I don't have any of these.
  3. If the option does not appear:
    • Try logging in at xbox.com
    • Use a United States VPN
    • Try a different browser or device

Use Recovery Code

  1. Paste the 25-character recovery code:
  2. XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
  3. Click Use recovery code.
  4. Change the password and update the security email.

Set Up Two-Step Verification

  1. Login again at account.live.com/proofs/manage.
  2. Add an authenticator app:
    • Microsoft Authenticator
    • Authy
    • Zyger 2FA
  3. Enable two-step verification.

Add a New Sign-In Method

  1. Click Add a new way to sign in or verify.
  2. Select Email a Code.
  3. Enter your email and retrieve the verification code.

Update Profile Information

  1. Go to account.microsoft.com/profile.
  2. Edit full name, date of birth, and country.
  3. Write this information down for account recovery.

Add a New Email Alias

  1. Click Edit account info.
  2. Select Add Email.
  3. Add a new Outlook email (recommended).
  4. Verify and set it as Primary.
  5. Remove the old alias.

Fully Securing the Account

  1. Remove all devices at account.microsoft.com/devices.
  2. Unlink devices at account.microsoft.com/devices/content.
  3. Remove app permissions at microsoft.com/consent.
  4. Leave Microsoft Family groups (if possible).
  5. Sign out everywhere and reset Windows Hello.
✓ You have now fully secured your MFA account.
