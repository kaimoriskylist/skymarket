MFA Secure Guide

This is the official Sky Market MFA security manual. Follow every step carefully. Skipping steps, rushing, or ignoring warnings can permanently lock your account. Security is not optional — it is mandatory.

1. Changing the Email

  1. Login to the account normally.
  2. Visit account.live.com/proofs/manage.
  3. Enter the provided email address.
  4. If prompted for a code, select Other ways to sign in.
Always double-check the email you enter. Typos cause account locks.

2. Recovering the Password

  1. Click Forgot password?.
  2. Select I don’t have any of these.
  3. If missing, try Xbox.com or a US VPN.

3. Using the Recovery Code

  1. Paste the 25-character recovery code.
  2. Confirm recovery.
  3. Change password immediately.
  4. Update security email.
Never share recovery codes. Sky Market staff will never ask for them.

4. Enabling Two-Step Verification

  1. Add an authenticator app.
  2. Enable two-step verification.
  3. Save backup codes offline.

5. Final Account Lockdown

  1. Remove all devices.
  2. Revoke app permissions.
  3. Sign out everywhere.
  4. Reset Windows Hello.
