MFA Secure Guide

This document explains how to fully secure your Microsoft / Minecraft account using MFA. Follow each step carefully and do not skip any section. Security mistakes are permanent and may result in account loss.

Changing the Email

  1. Login to the account.
  2. Go to account.live.com/proofs/manage.
  3. Enter the provided email address.
  4. If asked for a code, use Other ways to sign in.
Take your time here — rushing increases mistakes.

Password Recovery

  1. Select Forgot password?.
  2. Choose I don’t have any of these.
  3. If unavailable, try Xbox.com or a US VPN.

Using Recovery Code

  1. Paste the 25-character recovery code.
  2. Confirm and change the password.
  3. Update the security email immediately.

Two-Step Verification

  1. Add an authenticator app.
  2. Enable two-step verification.
  3. Save backup codes offline.

Final Lockdown

  1. Remove all devices.
  2. Revoke app permissions.
  3. Sign out everywhere.
✓ Your account is now fully secured with MFA.
